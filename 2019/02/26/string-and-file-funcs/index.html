string and file funcs

发表于 2019-02-26

字符串相关函数:
strcpy(dest,src)
//将src拷贝到dest

strncpy(dest,src,num)
//将src前num个字符拷贝到dest

strcmp(s1,s2)
//s1==s2时返回0，s1 > s2时返回整数，s1 < s2时返回负数 (字典序,ASCII码大小)

strcat(dest,src)
//将src接到dest后面，注意dest的结束符会被去掉

strncat(dest,src,num)
//将src的前num个字符接到dest后面

其他可能比较有用的：
strchr(const char *string, int c)//s1中查找c
strstr(s1,s2)//在s1中查找子串s2

另外，将字符串指针作为函数的实参，定义为 f(char **s)，调用需要传入指针的地址 f(&s)，具体的还有待实验和补充

文件相关函数：
FILE *fp=fopen(addr,”r”/“w”/“a”/…) //分别是read only,新建或清空,新建或附加…
fscanf(fp,””,…)
fprintf(fp,””,…)
别忘了fclose(fp)

文件中search的代码格式：
fscanf(fp,”%s”,s);
while (!feof(fp)){
…
fscanf(fp,”%s”,s);
}
